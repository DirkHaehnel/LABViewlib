AFM Mode

The AFM mode is used to record data without moving the piezo. Before the measurement, the objective must be aligned with the AFM in the main application. All stage movement should be disabled in the source code of Pick and Destroy.

SPC Setup

Stage Setup

When selecting the PI TCP/IP connection, the host address of the stage controller must be entered. The default can be changed by editing setupStage.vi, changing the numbers in the Host address control followed by a right click-->data operations-->make current value default.
