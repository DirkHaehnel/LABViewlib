Batch Analyze

Chose a header file and push the button.
For the first Scan, the windows "settings", "TAC window" and "Spotfinding Settings" will pop up, you can fill in appropriate values (and immediately check on the Scan image). After "Update" and "Done", the program will crawl through all the other Scans in the folder and analyze the data with the same settings.

some fun facts:

still only works if the folders are labeled like

.../1/1.xml.hdr
.../2/2.xml.hdr

you can use a random name, but, name of the folder and the header file need to match. Also they have to be numbered in increments, the starting number doesnt matter

for data of ingos setup it should work for both the "results.txt" (lifetimes, intensities, etc.) and the "decay of spotxxx.dat". For data from my setup, so far only the "results.txt" works, but that'S enough for you guillermo, right?

in order to see some useful images, you need to set the threshold to approriate values, before starting the analysis. So either load a scan and adjust, or put the values before loading the first scan (which will cause an error, but doesn't matter)

If you analyze the old-fashioned way, always use the TAC Window Button before the Spotfinder Button (for the first scan at least)
